# IT105 Midterm: Student Management System
**Developer:** Borleo, Arjiannah Carmelle A.

## Project Description
This project is a mini Information System Database designed to manage school records. It tracks students, instructors, courses, and enrollments, ensuring that data is organized and easily accessible.

## Database Tables
- **Instructors:** Stores faculty details (Name, Email, Department).
- **Courses:** Lists subjects and links them to the assigned instructor.
- **Students:** Stores student information and enrollment dates.
- **Enrollments:** The bridge table connecting students to courses and tracking grades.

## Features Implemented
- **Relational Structure:** 4 tables connected with Primary and Foreign Keys.
- **Data Integrity:** Fully normalized to 3rd Normal Form (3NF).
- **SQL Operations:** Demonstrated CRUD operations (SELECT, INSERT, UPDATE, DELETE), JOINS, and Subqueries.
- **Optimization:** Implemented indexing on the `LastName` column to improve search performance.
